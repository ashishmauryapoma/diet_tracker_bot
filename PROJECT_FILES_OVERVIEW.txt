================================================================================
                            WATER LOGGING FLOW
================================================================================

User sends message:
  "drank 250ml water"
            ↓
        ┌───────────────────────────────────┐
        │ handle_text() in handlers.py      │
        └───────────────────────────────────┘
            ↓
    Is user authenticated?
        ↓ No → Show password prompt
        ↓ Yes
    ┌───────────────────────────────────┐
    │ try_parse_water_ml(text)          │
    │ Returns: 250 or None              │
    └───────────────────────────────────┘
        ↓
    Water detected? (ml is not None)
        ↓ Yes
        │
        ├─ Create WaterEntry(250, logged_at)
        │  ↓ ValidationError? → Show error, return
        │
        ├─ Fetch today's water total from sheets
        │
        ├─ Format preview with format_water_confirmation()
        │  "💧 +250 ml logged
        │   🔵🔵🔵⚪⚪⚪⚪⚪
        │   Today: 1.25 L / 2.00 L goal
        │   Remaining: 750 ml"
        │
        ├─ Show inline keyboard with buttons:
        │  [✅ Log it] [❌ Discard]
        │
        ├─ Store in pending dict:
        │  context.user_data[_PENDING_WATER_KEY][msg_id] = entry
        │
        └─ RETURN (water handled, skip food detection)

    ↓ No water detected
    └─ Proceed with food detection (existing logic)

User clicks button:
  [✅ Log it]
        ↓
    ┌──────────────────────────────────────┐
    │ water_confirm_callback()             │
    │ in handlers.py                       │
    └──────────────────────────────────────┘
        ↓
    ├─ Check if user authenticated
    │
    ├─ Retrieve pending water entry
    │
    ├─ If WATER_CONFIRM_CALLBACK:
    │  ├─ append_water_log() to Water_Log sheet
    │  ├─ refresh_daily_summary() to update Daily_Summary
    │  └─ Show: "✅ +250 ml logged to your sheet!"
    │
    ├─ If WATER_UNDO_CALLBACK:
    │  └─ Show: "💧 Entry discarded."
    │
    └─ Handle errors:
       ├─ SheetsError → Show error in preview
       └─ Missing entry → Show "entry expired"

Result:
  ✓ Water_Log sheet has new row: [Date, Time, 250]
  ✓ Daily_Summary updated with new water total
  ✓ Water visible in /goal and /today commands
